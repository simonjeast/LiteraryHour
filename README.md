# Literary Time Landing Page

## Client
Sarah (student who wanted a simple literary-themed landing page for quick reading breaks).

## Project Description
This project is a one-page site built with HTML, CSS, and a small amount of JavaScript for dynamic behavior. The page centers the current time in bold text, shows a literary quote below it, and includes a direct link to the quote's source material.

## Features
- Live time display in the center of the page
- Popular literary quote shown below the time
- Hyperlink to the quote's source text
- Scenic background rotation tied to the selected quote
- Automatic page refresh at the top of each hour
- Small manual `Refresh now` button
- Mobile-friendly, responsive layout

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript (for time updates, quote/background selection, and refresh scheduling)


## File Overview
- `index.html` - semantic page structure
- `styles.css` - layout, typography, visual styling, and animation
- `script.js` - live clock, quote/background rotation logic, and refresh controls
- `PROPOSAL.md` - client brief and scoped build plan

## What I Learned
- How to combine a simple static layout with light dynamic behavior
- How to schedule a timed refresh for the next exact hour
- How to pair visual themes (backgrounds) with content (quotes) while keeping the code small
